Lab9 - Coding bat practice with loops

Background

Part 1: Account Setup(ignore if you did this in previous coding bat lab)

  1. Go to codingbat.com and create an account (click the link on the upper right). Please use your JMU email when setting up your account.
  2. After logging in, click the "prefs" link (upper right) and "share to" your professor's JMU e-mail for example: chaoaj@jmu.edu so you will get credit. Also make sure your name is entered as shown: Last, First

============

Part 2 Problem Solving

  1. Solve as many of the List-2 problems as you can today. Solve the following: count_evens ,big_diff, centered_average, sum13.

Submission

Package up the 4 solutions above into a program called batlist2.py and submit it to https://gradescope.com https://gradescope.com by midnight tomorrow evening.
